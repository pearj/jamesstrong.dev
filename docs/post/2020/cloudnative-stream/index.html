The Tech used

Panelist: Zoom meeting Breakouts Separate Zoom Meeting

Stream: Open Broadcast Software Polls: Restream with OBS browswer Plugin Twitter feed: https://twitterfall.com/ with OBS broswer plugin

Stream Source: OBS on my home setup Stream Destinations: Youtube Live and Periscope

The speaker and panelist joined zoom as I did too.

I used OBS to stream the zoom to restream, for the overlays, and playing the music.

OBS allowed me to integrate plugins from streamlabs for the polls, and Twitter feed.

Restream is what streamed to youtube and Twitter, and restream pulled the chats as well.

Audio setup

That is a bit more tricky especially on a Mac, windows it's a littler easier.

you need to have a plugin to capture the audio from Zoom.

On mac you would use this ishowu audio

https://obsproject.com/forum/resources/os-x-capture-audio-with-ishowu-audio-capture.505/

https://lofi-gaming.org.uk/blog/2016/09/17/capture-mac-desktop-audio-obs/

On Windows there is an OBS plugin available.

Pricing https://www.epidemicsound.com/ $15/month personal Streamlabs Prime $19/month Restream $41/month

I also have a streamdeck that helps with Scene transitions for OBS. $149 https://www.elgato.com/en/gaming/stream-deck https://www.amazon.com/Elgato-Stream-Deck-Controller-customizable/dp/B06XKNZT1P/
